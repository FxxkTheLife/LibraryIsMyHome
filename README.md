## Usage

Windows 运行 `console_start.bat`

Mac 运行 `console_start.command`

也可自行 `python console_start.py`

运行前请确保 `pip install -r requirements.txt`

iPhone 推荐使用 Pythonista 3, Android 推荐使用 Pydroid
